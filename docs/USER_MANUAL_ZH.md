# iFakeGPS 使用手冊 (User Manual)

歡迎使用 iFakeGPS！這是一款專為 iOS 設備設計的虛擬定位工具，支援 iOS 17+。

## 🎬 快速開始 (Quick Start)

### 1. 安裝前置軟體 (Prerequisites)
在使用本軟體之前，請確保您的電腦已安裝以下軟體：
*   **iTunes**: Apple 官方提供的 iTunes (建議從 Apple 官網下載，而非 Microsoft Store)。這是為了確保電腦能正確識別您的 iOS 設備。

### 2. 開啟開發者模式 (Enable Developer Mode)
**這是最重要的一步！** iOS 16 及更高版本需要開啟「開發者模式」才能使用虛擬定位功能。

**如何開啟：**
1.  拿起您的 iPhone/iPad，進入 **「設定」(Settings)**。
2.  點選 **「隱私權與安全性」(Privacy & Security)**。
3.  滑動到最底部，找到 **「開發者模式」(Developer Mode)** 選項。
4.  點擊進入並將開關 **打開**。
5.  系統會提示您重新啟動設備，請點選 **「重新啟動」(Restart)**。
6.  重啟後，解鎖螢幕，會跳出確認視窗，請點選 **「開啟」(Turn On)** 並輸入您的螢幕鎖定密碼。

### 3. 啟動程式 (Launch App)
*   為了確保程式能正常驅動 iOS 設備，請務必 **以系統管理員身分執行** `iFakeGPS.exe` (右鍵 -> 以系統管理員身分執行)。

### 4. 連接設備 (Connect Device)
1.  使用 USB 傳輸線將設備連接到電腦。
2.  若手機跳出「信任這部電腦？」(Trust this computer?)，請點選 **「信任」(Trust)** 並輸入密碼。
3.  在 iFakeGPS 程式左側，點擊 **「🔄」重新整理按鈕**。
4.  您的設備應該會出現在列表中，點擊它進行連接。

### 5. 開始飛！ (Start Spoofing)
*   **單點模式 (Single Point)**: 在地圖上點擊任意位置，您的手機定位將立即移動到該處。
*   **路線模式 (Route Mode)**:
    1.  切換到 "Route Mode"。
    2.  在地圖上點擊多個點來規劃路線。
    3.  調整 "Walking Speed" (走路速度)。
    4.  點擊 "Start" 開始模擬移動。

## ❓ 常見問題 (FAQ)

**Q: 為什麼找不到我的設備？ (Device not found)**
A: 
1. 確保已安裝 iTunes。
2. 確保已解鎖螢幕並信任電腦。
3. 確保程式是以 **系統管理員身分** 執行。
4. 嘗試重新插拔 USB 線。

**Q: 定位沒有改變？**
A: 請確認您已成功開啟「開發者模式」並重啟手機。如果連線狀態顯示已連接但定位無效，請嘗試斷開連接並重新連接。
